Torrent CentOS
===

## download box

[opscode-centos-7.0](http://opscode.github.io/bento/)

```
$ vagrant box torrent-centos http://opscode-vm-bento.s3.amazonaws.com/vagrant/virtualbox/opscode_centos-7.0_chef-provisionerless.box
```

